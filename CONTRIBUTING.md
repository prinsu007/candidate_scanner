# Contributing to Candidate Scanner

First off, thanks for taking the time to contribute! 🎉

### How to Contribute

1. **Fork** the repository on GitHub.
2. **Clone** the forked repository to your local machine.
3. **Create a new branch** for your feature or bug fix (`git checkout -b feature/amazing-feature`).
4. **Make your changes** and test them.
5. **Commit** your changes (`git commit -m 'Add some amazing feature'`).
6. **Push** to the branch (`git push origin feature/amazing-feature`).
7. **Open a Pull Request** against the `main` branch.

### Pull Request Guidelines
- Ensure your code follows the existing style.
- Update the `README.md` if your feature adds new functionality.
- Wait for a review from a maintainer (@prinsu007).
